​:codex-terminal-citation[codex-terminal-citation]{line_range_start=173 line_range_end=177 terminal_chunk_id=ゲーム仕様】
- 盤面は10x10。
- 勝利条件は通常の五目並べと同じで、縦・横・斜めのいずれかで5つ連続。
- プレイヤーは2人で交互に手番。
  - プレイヤーA: 黒になる確率 90%（白10%）
  - プレイヤーB: 黒になる確率 30%（白70%）
- 石を置いた直後は未観測の量子石（Q）として保持する。
- 未観測石には「黒になる確率（%）」を表示する（例: 90%, 30%）。

【表示仕様】
- 未観測石は「大きな丸」で表示し、丸の中に確率（%）を表示する。
- 丸の背景は黒↔白のグラデーションで、黒側の比率はその石の黒確率に一致させる。
- 観測後に確定した石は、黒は黒丸（●）、白は白丸（○）として大きく表示する。

【観測仕様】
- 「観測する」ボタンを用意する。
- 観測時、盤上の全未観測石をそれぞれの石が持つ確率で黒/白に確定させる。
- 観測回数は共通ではなく、先手・後手それぞれ5回まで（個別カウント）。
- 勝敗未決着時のみ「再開（確率表示に戻す）」ボタンを押すと、直前の観測前状態（確率表示の盤面）に戻せるようにする。

【勝敗判定】
- 観測後に黒のみ5連: 黒勝ち。
- 観測後に白のみ5連: 白勝ち。
- 観測後に黒白同時に5連: 観測した手番プレイヤーの勝ち（タイブレーク）。
- 双方が観測上限（各5回）に達しても勝者が出ない場合は引き分け。

【UI要件】
- 表示項目:
  - 手番表示
  - 観測残り回数（A/Bそれぞれ）
  - 勝者表示
  - ログ表示
- 操作ボタン:
  - 観測する
  - 再開（確率表示に戻す）
  - リセット
- 勝者確定後は不正な操作ができないように制御する。
